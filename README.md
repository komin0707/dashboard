# ACS Dashboard (공유폴더 실행용)

## 실행
- `index.html`을 브라우저에서 열면 동작합니다.
- 모바일/다른 PC에서 접속하려면 **반드시 이 저장소 폴더에서** 서버를 실행하세요.
  - 예시
    - `cd /path/to/dashboard`
    - `python3 -m http.server 8080 --bind 10.30.50.233`
    - 접속: `http://10.30.50.233:8080/index.html`

### 모바일에서 `Directory listing for /`가 보일 때
- 서버를 `index.html`이 없는 다른 폴더(예: 사용자 홈 폴더)에서 실행한 상태입니다.
- 또는 `/index.html`이 아닌 루트(`/`)로만 접속한 상태입니다.
- 위 실행 예시처럼 `dashboard` 폴더에서 서버를 띄우고, URL 끝에 `/index.html`을 붙여 접속하세요.

### `404 Not Found`가 보일 때
- 접속 주소가 잘못된 경우가 가장 많습니다. (`/dashboard`, `/main` 같은 경로는 이 프로젝트에 없습니다)
- 정적 파일 서버 기준 올바른 주소는 아래 둘 중 하나입니다.
  - `http://10.30.50.233:8080/`
  - `http://10.30.50.233:8080/index.html`
- 서버를 올린 PC에서 먼저 아래로 확인하세요.
  - `curl -I http://127.0.0.1:8080/index.html` → `HTTP/1.0 200 OK`가 떠야 정상입니다.

## 주요 기능
- 게시판(방명록형 메모) 추가/삭제
- 달력 + 선택한 날짜 일정 관리(다른 날짜도 추가/삭제 가능)
- 교수 필터(전체 설정)
- 다음날 환자 리스트 자동 초기화
- 동일 `등록번호+환자명` 재등장 시 이전 상세 기록 복원
- 월간 지표 입력 카드
- 버튼형 바로가기(매크로/논문/공유폴더/교육자료/계산기)
- 전체 설정에서 경로 항목은 폴더 선택 버튼으로 입력 가능(브라우저 정책상 폴더명 기준)
- `komin nu` 버튼 추가: 설정에서 프로그램 경로 저장 후 즉시 실행 가능
- `강제종료` 버튼 추가: 설정에서 실행 파일 경로 지정 후 실행 가능
- `스케줄` 버튼 추가: 스탭/인턴/외래/수술방 스케줄 이미지를 팝업에서 항목별 1장씩 등록/교체/삭제
- `환자교육` 버튼: 큰 팝업에서 교육 항목 추가/삭제 및 PDF 넓은 뷰어 지원 (설정에서 경로 등록 또는 PDF 직접등록, 항목별 새창 열기 가능)

## 주의
브라우저 보안 정책상 로컬 exe 실행 및 전역 자동 키입력은 제한될 수 있습니다.\
실제 완전 자동 로그인(프로그램 열기→ID/TAB/PW/ENTER)은 AutoHotkey/사내 런처/Electron 같은 데스크톱 연동 방식이 권장됩니다.

- 특히 iOS/안드로이드 모바일 브라우저는 `file://` 또는 `\\서버\공유폴더\...` 실행을 대부분 차단합니다.
- 공유폴더/실행파일 열기는 대시보드를 띄운 **같은 Windows PC 브라우저**에서 사용하는 것을 권장합니다.
