# ACS Dashboard (공유폴더 실행용)

## 실행
- `index.html`을 브라우저에서 열면 동작합니다.
- 고정 IP `10.30.50.233`에서 실행 예: `python3 -m http.server 8080 --bind 10.30.50.233` 후 `http://10.30.50.233:8080`

## 주요 기능
- 환자 리스트 + 상세 수정/삭제/저장
- 엑셀(xlsx/xls/csv) 불러오기 (1번째 시트 기준)
- 교수 필터(전체 설정)
- 다음날 환자 리스트 자동 초기화
- 동일 `등록번호+환자명` 재등장 시 이전 상세 기록 복원
- 달력 + 금일 일정 관리
- 게시판(방명록형 메모) 추가/삭제
- 월간 지표 입력 카드
- 버튼형 바로가기(매크로/논문/공유폴더/교육자료/계산기)
- 전체 설정에서 경로 항목은 폴더 선택 버튼으로 입력 가능(브라우저 정책상 폴더명 기준)
- `komin nu` 버튼 추가: 설정에서 프로그램 경로 저장 후 즉시 실행 가능
- `강제종료` 버튼 추가: 설정에서 실행 파일 경로 지정 후 실행 가능
- `스케줄` 버튼 추가: 스탭/인턴/외래/수술방 스케줄 이미지를 팝업에서 항목별 1장씩 등록/교체/삭제

## 주의
브라우저 보안 정책상 로컬 exe 실행 및 전역 자동 키입력은 제한될 수 있습니다.\
실제 완전 자동 로그인(프로그램 열기→ID/TAB/PW/ENTER)은 AutoHotkey/사내 런처/Electron 같은 데스크톱 연동 방식이 권장됩니다.
